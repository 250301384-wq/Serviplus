Stakeholder Analysis

Understanding the diverse needs of key stakeholders across the organization

Marketing Department

Key Requirements:

  • "Early Warning" alert system (45 days inactivity)
  • Proactive retention campaigns
  • Targeted re-engagement strategies

Finance Department

Key Requirements:

  • Financial impact analysis
  • Revenue loss projections
  • Budget adjustment capabilities

Customer Support

Key Requirements:

  • Individual customer risk profiles
  • Churn scores and reasons
  • Personalized conversation tools

Management Team

Key Requirements:

  • High-level strategic dashboards
  • Cross-service performance comparison
  • Strategic decision-making support

Compliance Team

Key Requirements:

  • GDPR adherence and data privacy
  • "Non-Black Box" AI model
  • Transparent, explainable results

Project Timeline

Phased implementation approach with 3 mini-sprints aligned with the V-Cycle phases

1

Sprint 1: Understanding & Alignment

Duration: 2 weeks | Focus: Resolve ambiguities and finalize data governance rules

Key Activities:

  • • Stakeholder workshops
  • • GDPR compliance assessment
  • • Hardware capability review
  • • Data governance framework

Deliverables:

  • • Approved churn definition
  • • GDPR assessment report
  • • Hardware analysis
  • • Project charter
2

Sprint 2: Baseline Model & Explanations

Duration: 4 weeks | Focus: Build the "Non-Black Box" logic and handle inconsistent data logs

Key Activities:

  • • Data cleaning & normalization
  • • GDPR-compliant data masking
  • • Predictive model architecture
  • • Feature importance mechanisms

Deliverables:

  • • Data processing pipelines
  • • GDPR-compliant system
  • • Predictive model
  • • Performance metrics
3

Sprint 3: User-Facing Outputs

Duration: 6 weeks | Focus: Deliver the final dashboards and automated alerting systems

Key Activities:

  • • Department-specific dashboards
  • • Automated alerting system
  • • Financial forecasting tools
  • • User acceptance testing

Deliverables:

  • • Marketing dashboard
  • • Finance dashboard
  • • Support dashboard
  • • Management dashboard
